使用说明

  1. 获取安全密钥 (PAT)：
    GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
    勾选权限：repo (全选) 和 workflow
  2. 填写配置信息：
    用户名：你的GitHub账号名
    仓库名称：默认值已填好，可修改
    安全密钥：粘贴复制的密钥
    点击 保存配置
  3. 触发任务：
    点击对应任务的"触发"按钮
    等待1-2分钟查看结果
